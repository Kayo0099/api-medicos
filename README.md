# 🏥 API de Médicos

API REST desenvolvida em **Java 17 + Spring Boot 3**, para cadastro e gerenciamento de médicos.  
Permite criar, listar, atualizar e desativar registros, seguindo boas práticas de REST.

---

## 🚀 Tecnologias
- Java 17
- Spring Boot 3
- Spring Data JPA
- Flyway (controle de migrações)
- MySQL / H2 Database
- Maven ou Gradle
